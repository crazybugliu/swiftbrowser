If you are using a recent Mac OS X (Lion or Later), please allow running unsigned applications.

To do so :

 - Open System Preferences > Security > General and check "Authorize applications downloaded from everywhere"

OR

 - You can simply click on the "System Settings.command" that will open the Security Panel, unlock the lock and select "Allow applications from anywhere".
